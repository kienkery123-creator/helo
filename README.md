# 👋 Xin chào, mình là Kiên!

Cảm ơn bạn đã ghé thăm GitHub của mình. Mình hiện là sinh viên năm 2 ngành **Công nghệ Thông tin (ICT)**,

##  Về mình
-  **Dự án đang thực hiện:** Phát triển các hệ thống Web sử dụng framework Laravel và ASP.NET Core MVC.
-  **Frontend:** Yêu thích sự gọn nhẹ và hiện đại của Tailwind CSS.
-  **Công cụ & Hệ thống:** Có kinh nghiệm quản trị hệ thống cơ bản, tối ưu hóa phần cứng và làm việc với Git/GitHub.
  
##  Kỹ năng chuyên môn
| Lĩnh vực | Công nghệ sử dụng |
| :--- | :--- |
| **Backend** | PHP (Laravel), C# (ASP.NET Core MVC) |
| **Database** | MySQL, SQL Server |

##  Sở thích khác
Ngoài lúc ngồi code, mình thường giải trí với:
* **Gaming:** Teamfight Tactics (TFT) và ARK.
* **Khám phá:** Tìm hiểu về Redstone trong Minecraft hoặc nghiên cứu các linh kiện phần cứng máy tính.
